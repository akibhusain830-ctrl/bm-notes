### PART B: Short Answer Questions (2 × 5 = 10 Marks)

**6. If A = [1 2; 3 4], show that (Aᵀ)ᵀ = A.**
> **Solution:**
> `Aᵀ = [1 3; 2 4]`.
> `(Aᵀ)ᵀ = [1 2; 3 4] = A`.
> **Hence proved.**

**7. State the Zero Row Property of determinants.**
> **Answer:** If all elements of any row or column of a determinant are zero, the value of the determinant is **zero**.

**8. Define Marginal Revenue (MR) and find it for TR = 10x - x².**
> **Solution:** `MR = d(TR)/dx = 10 - 2x`.
> **Answer: 10 - 2x**

**9. Differentiate y = log(sin x) with respect to x. (Chain Rule)**
> **Solution:** `dy/dx = (1 / sin x) × cos x = cot x`.
> **Answer: cot x**

**10. If A = [2 0; 0 2], show that A is a scalar matrix.**
> **Solution:** Since it is a diagonal matrix where both diagonal elements are equal (2), it satisfies the definition of a **Scalar Matrix**.

**11. Solve using Cramer's Rule: [x+y+z=3; 2x-y+z=2; x-2y+3z=2]**

> 1. `D = |1 1 1; 2 -1 1; 1 -2 3| = 1(-1) - 1(5) + 1(-3) = -9`.

**12. Find the inverse of A = [1 2 1; 3 2 3; 1 1 2].**
> **Solution Steps:**
> 1. `|A| = 1(4-3) - 2(6-3) + 1(3-2) = 1 - 6 + 1 = -4`.
> 2. **Cofactors:** `C₁₁=1, C₁₂=-3, C₁₃=1, C₂₁=-3, C₂₂=1, C₂₃=1, C₃₁=4, C₃₂=0, C₃₃=-4`.
> 3. `Adj A = [1 -3 4; -3 1 0; 1 1 -4]`.
> 4. `A⁻¹ = Adj A / -4`.
> **Answer: -1/4 [1 -3 4; -3 1 0; 1 1 -4]**

**13. LPP Min Z = 20x + 10y; Sub to: x + 2y ≥ 40; 3x + y ≥ 30; 4x + 3y ≥ 60; x, y ≥ 0.**
> **Solution Steps:**
> 1. Corner points of the feasible region (above the lines) need to be identified.
> 2. Intersection of x+2y=40 and 3x+y=30: `x=4, y=18`.
> 3. Other intersections: (0,30), (40,0).
> 4. `Z(0,30)=300, Z(40,0)=800, Z(4,18)=80+180=260`.
> **Answer: Min Z = 260 at x=4, y=18**

**14. TC = 5x² + 20x + 500. Find MC, AC and show MC > AC for x > 10.**
> **Solution Steps:**
> 1. `MC = 10x + 20`.
> 2. `AC = 5x + 20 + 500/x`.
> 3. `MC - AC = (10x + 20) - (5x + 20 + 500/x) = 5x - 500/x`.
> 4. For `x > 10`: `5x > 50` and `500/x < 50`.
> 5. Therefore `5x - 500/x > 0` → `MC > AC`.
> **Hence proved.**
